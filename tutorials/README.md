# Tinker Tutorials

Hands-on guides for building with the Tinker SDK. Each tutorial walks through a different aspect of creating, fine-tuning, and deploying language models with Tinker, complementing the end-to-end scientific code generation pipeline in the project root.

## Available Tutorials

- [Getting Started with the Service Client](01_service_client_basics.md)
- [LoRA Fine-Tuning Workflow](02_lora_fine_tuning.md)
- [Evaluation and Monitoring](03_evaluation_and_monitoring.md)
- [Data Preparation Patterns](04_data_preparation.md)

## How to Use This Folder

1. Start with the service client basics to authenticate and explore models.
2. Follow the LoRA fine-tuning walkthrough alongside `train_scientific_code_model.py`.
3. Use the evaluation guide to validate and benchmark your runs.
4. Extend your datasets with the data preparation patterns.

Feel free to add more tutorials; just drop a new Markdown file here and link it in the list above.
